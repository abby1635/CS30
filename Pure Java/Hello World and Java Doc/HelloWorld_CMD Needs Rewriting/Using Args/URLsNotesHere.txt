A note from the website
System.exit(0) means it is a normal exit from a program.
But System.exit(-1) means the exit may be due to some error. 
Any number other that zero means abnormal exit. 
The parameter of System.exit(int) is the return value of your program, 
which can be evaluated in batch jobs (usually for console programs).
